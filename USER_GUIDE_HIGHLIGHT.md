## How to Use

### Basic Highlighting

**Step 1: Start Annotation**
1. Click the Screenshot & Annotate extension icon
2. Click "üì∏ Capture Screenshot"
3. Select the area you want to capture

**Step 2: Activate Highlight Tool**
- Click **üñçÔ∏è Highlight** button, OR
- Press **'H'** on your keyboard

**Step 3: Draw Highlights**
1. Your cursor changes to a crosshair (+)
2. Click and hold the mouse button
3. Drag to draw your highlight stroke
4. Release to complete the stroke

**Step 4: Continue Highlighting**
- Draw as many highlight strokes as you need
- Each stroke is independent
- Switch colors between strokes using the color picker

### Changing Colors

1. Locate the **color picker** in the toolbar (circular button with current color)
2. Click on it to open the color selector
3. Choose your desired color
4. Draw new highlights with the selected color

### Switching Between Tools

You can quickly switch between tools:
- **'H'** - Highlight tool
- **'P'** - Pen tool (for drawing)
- **'T'** - Text tool (for adding text)
- **'B'** - Blur tool (for censoring)
- **'V'** - Select tool (for moving/editing)

### Undoing Mistakes

- Press **Ctrl+Z** (Windows/Linux) or **Cmd+Z** (Mac)
- This removes the last highlight stroke
- Press multiple times to undo multiple strokes

### Saving Your Work

1. Click the **üíæ Save** button in the toolbar
2. Your annotated screenshot downloads automatically
3. File format: PNG with full quality
4. Highlights are permanently embedded in the image

## Tips & Best Practices

### üìå Pro Tips

**1. Highlighting Text**
- Draw slowly for straight lines along text
- Use yellow for best readability
- Make multiple passes for darker emphasis

**2. Emphasizing Areas**
- Use circular motions to highlight regions
- Different colors for different priorities
- Layer highlights for darker shading

**3. Creating Visual Hierarchy**
- Yellow: Most important information
- Green: Secondary points or correct answers
- Red: Errors or critical issues
- Blue: References or links

**4. Performance**
- Shorter strokes render faster
- Lift mouse between strokes for better performance
- Use Ctrl+Z immediately if you make a mistake

### ‚ö†Ô∏è Things to Note

**Highlights Cannot Be Edited After Drawing**
- Once drawn, highlights cannot be moved or resized
- They are permanent like pen strokes
- Plan your highlights before drawing
- Use Ctrl+Z immediately if you make a mistake

**Highlight Order Matters**
- Highlights are drawn in the order you create them
- Later highlights appear on top of earlier ones
- This affects the final appearance when overlapping

## Troubleshooting

### Highlight not appearing?
- Make sure the Highlight tool is active (button should be highlighted)
- Check that you're clicking and dragging, not just clicking
- Try pressing 'H' to reactivate the tool

### Highlight is too thin?
- The default thickness is 3x the pen stroke width
- This is intentional for the highlighter effect
- Future updates will include adjustable thickness

### Color not changing?
- Make sure you select a new color BEFORE drawing
- Already-drawn highlights keep their original color
- The color picker only affects new highlights

### Cannot move or resize highlights?
- This is expected behavior
- Highlights are permanent once drawn (like pen strokes)
- Use the Select tool ('V') for annotations that can be edited
- Use shapes (rectangles, circles) if you need resizable highlights

### Highlights look too light?
- The 40% opacity is intentional for realistic highlighting
- Draw multiple strokes over the same area for darker emphasis
- Or use the Pen tool for fully opaque drawing

## FAQ

**Q: Can I adjust the transparency?**
A: Currently fixed at 40%. Adjustable transparency is planned for future updates.

**Q: Can I make highlights thinner or thicker?**
A: Currently uses automatic thickness (3x pen width). Custom thickness coming in future updates.

**Q: Can I move a highlight after drawing it?**
A: No, highlights are permanent like pen strokes. Use Ctrl+Z to undo if needed.

**Q: What's the difference between highlight and pen?**
A: Highlights are semi-transparent and thicker, while pen strokes are opaque and thinner.

**Q: Can I use custom colors?**
A: Yes! Use the color picker to select any color you want.

**Q: Do highlights work on all websites?**
A: Yes, highlights work on any screenshot captured by the extension.

**Q: Are highlights saved in the final image?**
A: Yes, highlights are permanently embedded when you save the screenshot.
